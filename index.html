JEU DE CHASSE A LA MOUCHE

AVERTISSEMENT : Ce jeu est déconseillé au public sensible ou aux personnes de moins de 10 ans.

REGELEMENT DU JEU :
Objectif du jeu : Le joueur doit écraser 10 mouches. Déroulement du jeu : Le jeu se déroule sur une interface où des mouches apparaissent aléatoirement. Le joueur doit cliquer sur les mouches pour les écraser. Chaque mouche écrasée rapporte un point. Durée du jeu : Le jeu dure jusqu'à l'écrasement des 10 mouches. Fin du jeu : Le jeu se termine lorsque le joueur a écrasé 10 mouches. Si le joueur n'écrase pas les 10 mouches, la partie continue. Score : Le score du joueur est calculé en fonction du nombre de mouches écrasées. Gagnant : Le joueur qui écrase les 10 mouches est déclaré gagnant. Équipement : Le joueur peut utiliser uniquement la souris ou l'écran tactile pour écraser les mouches.
